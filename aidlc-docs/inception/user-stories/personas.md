# Personas
# 배달비 절약을 위한 음식 공동 구매 앱

---

## Persona 1: 방장 — 배달 주도형

| 항목 | 내용 |
|------|------|
| **이름** | 김민준 |
| **직업/상황** | 대학교 기숙사 거주 대학생 (2학년) |
| **위치 맥락** | 제2 기숙사, 정문 인근, 서천동 등 캠퍼스 근거리 |
| **목표** | 혼자 배달을 시키면 배달비(3,000~5,000원)가 음식값의 절반. 근처 사람들과 공동 주문하여 배달비를 나누고 싶음 |
| **불편함** | 카카오톡 단체방으로 공동주문을 조율하는 것이 번거롭고, 메뉴 취합/정산이 복잡함 |
| **행동 패턴** | 먹고 싶은 식당 결정 → 앱에서 방 개설 → 참여자 모집 → 주문 취합 → 배달비 정산 |
| **대표 스토리** | EP-02, EP-07, EP-08 |

## Persona 2: 방장 — 모임 주도형

| 항목 | 내용 |
|------|------|
| **이름** | 이지은 |
| **직업/상황** | 오피스텔 거주 직장인 (신입사원) |
| **위치 맥락** | 회사 근처 오피스텔, 퇴근 후 혼밥이 싫음 |
| **목표** | 배달 시켜서 같이 먹거나, 근처 식당에 다 같이 가고 싶음. 새로운 동네 사람과도 어울리고 싶음 |
| **불편함** | 함께 먹을 사람을 구하는 방법이 없음. SNS 채널은 너무 공개적이고 부담스러움 |
| **행동 패턴** | 모임 유형 선택(배달+모여먹기 or 직접외식) → 방 개설 → 참여자 모집 → 모임 진행 |
| **대표 스토리** | EP-02, EP-03, EP-07 |

## Persona 3: 참여자 — 메뉴 결정형

| 항목 | 내용 |
|------|------|
| **이름** | 박승현 |
| **직업/상황** | 대학원생, 연구실 근처 거주 |
| **위치 맥락** | 캠퍼스 인근, 혼밥 잦음 |
| **목표** | 오늘 먹고 싶은 음식은 정해져 있음. 같은 메뉴를 공동 주문하는 방을 빨리 찾고 싶음 |
| **불편함** | 원하는 식당의 방이 있는지 찾기 어렵고, 매번 새 방을 만들기엔 귀찮음 |
| **행동 패턴** | 앱 탐색(지도/검색) → 식당 필터링 → 방 찾기 → 참여 → 메뉴 선택 |
| **AI 추천** | 사용 안 함 (이미 먹고 싶은 것 결정) |
| **대표 스토리** | EP-04, EP-05 |

## Persona 4: 참여자 — 메뉴 미결정형

| 항목 | 내용 |
|------|------|
| **이름** | 최하은 |
| **직업/상황** | 대학교 학부생, 기숙사 거주 |
| **위치 맥락** | 제2 기숙사 주변 |
| **목표** | 배달 시키고 싶은데 뭘 먹을지 모름. AI 추천을 통해 메뉴를 결정하고 맞는 방을 찾고 싶음 |
| **불편함** | 선택지가 너무 많아서 결정 장애. 혼자 생각하면 결국 아무것도 못 고름 |
| **행동 패턴** | 앱 접속 → AI 추천 요청 (기호/위치 입력) → 추천 메뉴/식당 확인 → 관련 방 탐색 → 참여 또는 직접 개설 |
| **AI 추천** | 핵심 기능으로 활용 |
| **대표 스토리** | EP-04, EP-06 |

## S-01: 스플래시 화면

**목적**: 앱 로딩 + 인증 상태 판단 후 라우팅

```
+---------------------------+
|                           |
|                           |
|       [앱 로고]           |
|    배달 같이해요           |
|    "배달비, 같이 아끼자"  |
|                           |
|      [로딩 인디케이터]     |
|                           |
+---------------------------+
```

**주요 요소**

| 요소 | 타입 | 설명 |
|------|------|------|
| 앱 로고 | Image | 512x512 PNG, 중앙 배치 |
| 슬로건 | Text | 폰트 16px, 서브 컬러 |
| 로딩 인디케이터 | ActivityIndicator | 브랜드 컬러 |

**동작 로직**
- AsyncStorage에 닉네임 존재 → `/home` 이동
- 닉네임 없음 → `/onboarding` 이동
- 표시 시간: 최소 1.5초, API 응답 대기 시 연장

## S-02: 닉네임 등록 화면

**목적**: 최초 1회 닉네임 및 학교 설정

```
+---------------------------+
| < 뒤로 (없음, 최초 설정)  |
+---------------------------+
|                           |
|   안녕하세요!             |
|   닉네임을 설정해주세요   |
|                           |
|  [닉네임 입력 TextField]  |
|   2~10자, 한/영/숫자      |
|                           |
|  [학교 선택 드롭다운]     |
|   예: 경희대학교          |
|                           |
|  [시작하기 버튼 - CTA]    |
|                           |
+---------------------------+
```

**주요 요소**

| 요소 | 타입 | 설명 |
|------|------|------|
| 닉네임 TextField | TextInput | maxLength=10, 실시간 유효성 검사 |
| 유효성 안내 | Text | 오류 시 빨간색 문구 |
| 학교 선택 | Picker / BottomSheet | 학교명 검색 가능 |
| 시작하기 버튼 | PrimaryButton | 유효성 통과 시 활성화 |

**유효성 규칙**
- 닉네임: 2~10자, 특수문자 제외
- 학교: 필수 선택
- 두 조건 모두 충족 시 버튼 활성화

## S-03: 홈 화면 (방 목록 + 지도)

**목적**: 방 탐색의 메인 진입점 — 지도/목록 듀얼 뷰

```
+---------------------------+
| [위치 아이콘] 경희대 주변  |  [알림] [프로필]
+---------------------------+
| [검색바: "식당명 검색..."] |
+---------------------------+
| [필터 칩 스크롤]          |
|  전체 | 한식 | 중식 | 분식|
+---------------------------+
|                           |
|       [지도 영역]         |  <- 탭 1
|   [D][G]  [D]  [R]       |
|       [D]                 |
|                           |
+---------------------------+
|   하단 시트 (방 목록)     |  <- 드래그 업
|  +-----------------------+|
|  | RoomCard              ||
|  | RoomCard              ||
|  +-----------------------+|
+---------------------------+
|   [+방 개설]              |  <- FAB 버튼
+---------------------------+
```

**지도 탭 요소**

| 요소 | 타입 | 설명 |
|------|------|------|
| 지도 | MapView (react-native-maps) | 현재 위치 중심 |
| 방 마커 | CustomMarker | [D]/[G]/[R] 유형별 아이콘 |
| 마커 탭 | BottomSheet | 방 미리보기 카드 노출 |
| 필터 칩 | ScrollView + Chip | 음식 카테고리 필터 |
| FAB | FloatingActionButton | 방 개설 진입 |

**목록 탭 요소**

| 요소 | 타입 | 설명 |
|------|------|------|
| 방 카드 리스트 | FlatList + RoomCard | 마감 임박 순 정렬 |
| 빈 상태 | EmptyState | "근처에 열린 방이 없어요. 방을 개설해보세요!" |
| 새로고침 | RefreshControl | Pull-to-refresh |

**방 마커 범례**
- `[D]` 공동배달 — 주황색
- `[G]` 배달+모여먹기 — 초록색
- `[R]` 직접외식 — 파란색

## S-04: 방 개설 화면

**목적**: 방장이 새 방을 설정하고 생성

```
+---------------------------+
| < 방 개설                 |
+---------------------------+
| 모임 유형 선택            |
| [공동배달][배달+모임][외식] |  <- 세그먼트 컨트롤
+---------------------------+
| 식당 이름 *               |
| [입력 필드]               |
|                           |
| 배달 주소 *               |
| [주소 검색]               |
|                           |
| 최소 주문금액             |
| [숫자 입력]               |
|                           |
| 배달비                    |
| [숫자 입력]  1인당: N원   |  <- 실시간 계산
|                           |
| 최대 인원 *               |
| [스텝퍼: 2~8명]           |
|                           |
| 마감 시간 *               |
| [시간 선택 Picker]        |
|                           |
| (유형에 따라 추가 필드)   |
| - 배달+모임: 모일 장소    |
| - 직접외식: 예상 출발시간 |
+---------------------------+
| [방 개설하기]             |
+---------------------------+
```

**주요 요소**

| 요소 | 타입 | 설명 |
|------|------|------|
| 유형 선택 | SegmentedControl | 공동배달 / 배달+모여먹기 / 직접외식 |
| 식당 이름 | TextInput | 필수 |
| 배달 주소 | AddressSearch | 주소 API 연동 (카카오 우편번호) |
| 배달비 입력 | NumericInput | 실시간 1인당 배달비 계산 표시 |
| 최대 인원 | Stepper | 2~8명 범위 |
| 마감 시간 | DateTimePicker | 현재 시간 이후만 선택 가능 |
| 공유 버튼 (생성 후) | ShareButton | 카카오톡 / 링크복사 / QR |

**조건부 필드**

| 유형 | 추가 필드 |
|------|-----------|
| 배달+모여먹기 | 모일 장소 주소 (필수) |
| 직접외식 | 방문 식당 주소, 예상 출발 시간 |

## S-05: 방 상세 화면

**목적**: 방 정보 확인 + 참여 결정

```
+---------------------------+
| < 뒤로    [공동배달] 뱃지 |
+---------------------------+
| [식당 대표 이미지]        |
+---------------------------+
| 홍콩반점 치킨마요 덮밥    |
| 배달 주소: 기숙사 1동 앞  |
| 마감까지: 00:23:41 ⏱     |
+---------------------------+
| 방장: 민준이🍜           |
+---------------------------+
| 참여자 (3/5)              |
| [ParticipantList]         |
| O O O _ _                 |
+---------------------------+
| 배달비 정보               |
| 총 배달비: 5,000원        |
| 현재 1인당: 1,667원 →    |
| 가득 차면: 1,000원        |
+---------------------------+
| 최소 주문금액: 15,000원   |
+---------------------------+
| [채팅 보기]  [참여하기]   |
+---------------------------+
```

**주요 요소**

| 요소 | 타입 | 설명 |
|------|------|------|
| 유형 뱃지 | Badge | 공동배달 / 배달+모여먹기 / 직접외식 |
| 식당 이미지 | Image | CloudFront CDN, 16:9 비율 |
| 마감 카운트다운 | CountdownTimer | 실시간 업데이트 |
| 참여자 목록 | ParticipantList | 아바타 나열, 빈 슬롯 표시 |
| 배달비 계산 | DeliveryFeePreview | 현재 인원 / 만석 시 각각 표시 |
| 참여하기 버튼 | PrimaryButton | 만석 시 비활성화 |
| 채팅 보기 버튼 | SecondaryButton | 참여 전에도 읽기 가능 (선택) |
| 공유 버튼 | IconButton | 상단 우측 |

**방 상태별 버튼 변화**

| 상태 | 버튼 표시 |
|------|-----------|
| 모집중 | "참여하기" (활성) |
| 만석 | "참여하기" (비활성) + "대기 알림 받기" |
| 주문완료 | "정산 보기" |
| 완료 | "후기 남기기" |

## S-06: 메뉴 선택 화면

**목적**: 참여자가 자신의 메뉴를 선택/입력

```
+---------------------------+
| < 뒤로    메뉴 선택       |
+---------------------------+
| [배달앱 메뉴 불러오기 성공]|
+---------------------------+
| [메뉴 카테고리 탭]        |
| 메인 | 사이드 | 음료      |
+---------------------------+
| [OrderItemRow]            |
| 치킨마요 덮밥   8,500원 [+]|
| [OrderItemRow]            |
| 돈까스 덮밥    8,000원 [+]|
| [OrderItemRow]            |
| 단무지 추가      500원 [+]|
+---------------------------+
| -- 또는 직접 입력 --      |
| [+ 메뉴 직접 추가]        |
+---------------------------+
| 내 주문 요약              |
| 치킨마요 덮밥 x1  8,500원 |
| 단무지 추가   x1    500원 |
+---------------------------+
| 소계: 9,000원             |
| + 1인 배달비: 1,000원     |
| = 예상 합계: 10,000원     |
+---------------------------+
| [주문 완료]               |
+---------------------------+
```

**주요 요소**

| 요소 | 타입 | 설명 |
|------|------|------|
| 메뉴 카테고리 탭 | TabBar | API 응답 카테고리 기반 |
| 메뉴 리스트 | FlatList + OrderItemRow | 수량 조절 (+/-) 포함 |
| 직접 입력 폼 | Modal | 메뉴명 / 가격 / 수량 |
| 주문 요약 | StickyFooter | 소계 + 배달비 + 합계 실시간 업데이트 |
| 주문 완료 버튼 | PrimaryButton | 1개 이상 선택 시 활성화 |

**메뉴 불러오기 상태**

| 상태 | 표시 |
|------|------|
| 로딩 중 | 스켈레톤 UI |
| 성공 | 메뉴 리스트 표시 |
| 실패 | "직접 입력" 모드로 전환 안내 |

## S-07: 정산 화면

**목적**: 최종 결제 금액 확인 + 송금 안내

```
+---------------------------+
| 정산 내역                 |
+---------------------------+
| 내 주문 항목              |
| [OrderItemRow - 읽기전용] |
| 치킨마요 덮밥   8,500원   |
| 단무지 추가       500원   |
+---------------------------+
| 소계:           9,000원   |
| 1인 배달비:     1,000원   |
+---------------------------+
| 최종 금액:     10,000원   |
| (굵고 크게)               |
+---------------------------+
| 송금 계좌 (방장)          |
| 카카오뱅크                |
| 1234-5678-9012            |
| 예금주: 김민준            |
| [계좌번호 복사]           |
+---------------------------+
| [SettlementSummary]       |
| 전체 참여자 현황          |
| 민준이 ✅ 완료            |
| 지은이 ⏳ 대기            |
| 승현이 ✅ 완료            |
+---------------------------+
| [송금 완료 표시]          |
+---------------------------+
```

**주요 요소**

| 요소 | 타입 | 설명 |
|------|------|------|
| 주문 항목 | OrderItemRow (읽기 전용) | 수정 불가 상태 |
| 최종 금액 | Text (Large/Bold) | 소계 + 1인 배달비 합산 |
| 계좌 복사 | TouchableOpacity | 클립보드 복사 + 토스트 |
| 참여자 정산 현황 | SettlementSummary | 방장만 전체 현황 보임 |
| 송금 완료 버튼 | PrimaryButton | 탭 시 내 상태 완료 처리 |

**방장 전용 요소**
- 전체 참여자 송금 완료 여부 확인
- "배달 완료" 버튼 (배달중 상태에서만 노출)

## S-08: 채팅 화면

**목적**: 방 참여자 간 실시간 소통

```
+---------------------------+
| < 뒤로  홍콩반점 치킨방   |
|         참여자 3명        |
+---------------------------+
|                           |
| [시스템] 지은이가 참여했습니다
|                           |
| [민준이] 메뉴 선택해주세요~
|          14:32            |
|                           |
|      나: 치킨마요 골랐어요!
|                     14:33 |
|                           |
| [시스템] 승현이가 메뉴를 선택했습니다
|                           |
| [민준이] 그럼 주문할게요!
|          14:35            |
|                           |
| [시스템] 주문이 확정되었습니다
|                           |
+---------------------------+
| [메시지 입력...]  [전송]  |
+---------------------------+
```

**주요 요소**

| 요소 | 타입 | 설명 |
|------|------|------|
| 메시지 리스트 | FlatList (역방향) | 최신 메시지 하단 고정 |
| 내 메시지 | ChatBubble (우측 정렬) | 브랜드 컬러 배경 |
| 상대 메시지 | ChatBubble (좌측 정렬) | 회색 배경 + 닉네임 |
| 시스템 메시지 | SystemMessage | 중앙 정렬, 작은 폰트 |
| 입력창 | TextInput | 멀티라인 지원, 키보드 회피 |
| 전송 버튼 | IconButton | 텍스트 입력 시 활성화 |

**시스템 메시지 유형**
- `{닉네임}님이 참여했습니다`
- `{닉네임}님이 퇴장했습니다`
- `{닉네임}님이 메뉴를 선택했습니다`
- `주문이 확정되었습니다`
- `배달이 완료되었습니다`

**방 종료 후**: 입력창 비활성화, "종료된 방입니다" 안내 배너 표시
